## Frequently Asked Questions (FAQ)

Find answers to common questions about our Minecraft AI Bots below.

### General Questions

**Q: What can the bot do?**

A: The bot can perform a variety of tasks, including movements, mining, crafting, farming, fighting, teleporting, and building structures. For more details, see the [Capabilities](/capabilities) page.

**Q: How do I interact with the bot?**

A: We recommend using the whisper feature to communicate with the bot. You can also interact with other players' bots by mentioning their names in world chat if you are close to them.

**Q: Can the bot build any structure I want?**

A: Currently, the bot can build a predefined list of structures. We are working on a feature that will allow you to ask the bot to build anything you want.

### Installation and Setup

**Q: How do I install the client?**

A: You can download the client for Windows and Mac from our website. Follow the installation instructions provided in the [Launcher Guide](/launcher).

**Q: Do I need to use the client to use the bot?**

A: While using the client is recommended for the best experience, you can also connect to the server using a standard Minecraft Java 1.20.2 client. The server address is `public-beta.mc.elefant.gg`.

### Troubleshooting

**Q: What should I do if the bot gets stuck?**

A: If your bot gets stuck, you can ask it to stop or reconnect yourself to the server. Using whisper to communicate with the bot can also help.

**Q: How do I report an issue?**

A: You can report issues by posting in the support channel on our [Discord server](https://discord.gg/elefant).

### Additional Features

**Q: Can I customize the bot's appearance?**

A: Yes, you can change the agent character by selecting which character you want for each slot. You can summon up to 3 characters.

**Q: What are the additional options available for the bot?**

A: You can adjust the volume, enable or disable text-to-speech and speech-to-text features, and use the open mic option. For more details, see the [Extra Options](/launcher#extra-options) section.

### Feedback and Support

**Q: How can I provide feedback?**

A: You can provide feedback by joining our [Discord server](https://discord.gg/elefant) and posting in the designated channels.
